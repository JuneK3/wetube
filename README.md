# Wetube
Youtube Clone Project with Nomad Coders

## Stacks
- Node, Express
- Pug, Scss (일단 완성후 React, Redux로 업데이트 예정)
- webpack
- passport.js
- MongoDB, mongoose

## Implements
- Users
  - User Authentication
  - Facebook Login
  - Github Login
  - User Profile
  - Log In
  - Log Out
  
- Videos
  - Video Upload
  - Video Recording
  - Search Video
  - AJAX Comments
  - View Count
